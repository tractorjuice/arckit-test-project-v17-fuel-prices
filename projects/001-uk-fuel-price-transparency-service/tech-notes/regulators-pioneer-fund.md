# Regulators' Pioneer Fund

## Overview

The Regulators' Pioneer Fund (RPF) is a DSIT-administered competitive fund that pays UK regulators and local authorities to run innovation projects. It aims to create a regulatory environment that fosters business innovation and investment.

## Key Facts

- **Administered by**: DSIT (Department for Science, Innovation and Technology)
- **Fund size**: ~£5.5M per round (Round 4)
- **Project funding**: £50,000 - £1,000,000 per project
- **Duration**: 6-12 month projects
- **Eligibility**: UK regulators and local authorities only
- **Contact**: regulators.pioneerfund@dsit.gov.uk

## Round 4 Details

- Applications closed 31 July 2025
- Funding period: 1 October 2025 - 30 September 2026
- 16 projects selected covering data protection, property market reform, startup navigation, unmanned aircraft, and more

## Relevance to Fuel Finder

CMA qualifies as a statutory regulator, making this the strongest-fit external funding source. Potential application areas:

- **Enforcement innovation**: Real-time compliance monitoring using Fuel Finder API data, automated non-compliance detection and escalation workflows
- **Open data innovation**: Enhanced API enabling third-party business innovation (navigation apps, price comparison services)
- **Regulatory technology**: Anomaly detection for price manipulation, geographic compliance heatmaps

## Application Process

1. Download application form from GOV.UK when round opens
2. Complete application demonstrating regulatory innovation
3. Submit to regulators.pioneerfund@dsit.gov.uk
4. Assessment and selection by DSIT panel
5. Funding awarded for project duration
