## Summary

PostGIS is the open source geospatial extension for PostgreSQL, providing spatial data types, indexing (GiST), and functions for proximity search, distance calculation, and geographic data processing. It is the recommended technology for implementing citizen fuel price search by location (FR-004) in the UK Fuel Price Transparency Service.

## Key Findings

- **Licence**: GPLv2 (PostgreSQL extension); compatible with AWS RDS managed hosting
- **Maturity**: Production-ready; widely deployed across UK Government (multiple local authority and central government implementations found in govreposcrape)
- **AWS RDS Support**: PostGIS is a supported extension on Amazon RDS for PostgreSQL
- **Key Functions for Fuel Finder**:
  - `ST_DWithin(geography, geography, distance_metres)` -- proximity search within radius
  - `ST_Distance(geography, geography)` -- calculate distance between two points
  - `ST_AsGeoJSON(geometry)` -- export to GeoJSON format (open data)
  - `ST_MakePoint(longitude, latitude)` -- create point from coordinates
- **Spatial Indexing**: GiST (Generalised Search Tree) index on geography columns enables sub-millisecond proximity queries across 10,000+ points
- **Performance**: Proximity queries across 8,500 forecourts with GiST index expected to return in < 50ms, well within the < 3 second p95 target (NFR-P-001)
- **UK Government Usage**: osmuk2pgsql project provides PostgreSQL+PostGIS database of UK OpenStreetMap data; multiple council GIS implementations found
- **Related Project**: Pumperly -- open source fuel and EV route planner using PostGIS spatial queries for finding cheapest stations along routes

## Relevance to Projects

- **Project 001 (UK Fuel Price Transparency Service)**: Core technology for citizen fuel price search by location. PostGIS handles proximity queries, distance calculations, and GeoJSON output for the open data API.
